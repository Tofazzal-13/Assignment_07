# 🌿 KeenKeeper

**KeenKeeper** is a personal relationship management tool designed to help you nurture and maintain meaningful connections. Just like a garden, relationships need tending—KeenKeeper ensures you never lose touch with the people who matter most.

---

## 🚀 Technologies Used

This project was built with a modern tech stack focused on performance and clean UI:

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (Pure CSS approach, No DaisyUI)
* **Icons:** React Icons
* **Charts:** Recharts (For Friendship Analytics)
* **Notifications:** React Hot Toast
* **State Management:** React Context API & Custom Hooks
* **Routing:** React Router v6

---

## ✨ Key Features

### 1. Personal Relationship Shelf 🗃️
Easily browse your circle of friends and family. Each contact has a dedicated details page where you can see the history of your interactions, set relationship goals, and view their specific status (like "Overdue" or "Family").

### 2. Quick Check-In & Timeline ⏳
Log interactions instantly! With just one click on the **Call, Text, or Video** buttons, KeenKeeper automatically generates a timestamped entry in your **Timeline**. It keeps a chronological history of how you’ve been showing up for your loved ones.

### 3. Friendship Analytics 📊
Gain insights into your social habits with a visual **Analytics Dashboard**. Using interactive Pie Charts, you can see the distribution of your interaction types (Calls vs. Texts vs. Videos) to ensure you are connecting in balanced and meaningful ways.

## 🛠️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/keenkeeper.git](https://github.com/your-username/keenkeeper.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the project:**
    ```bash
    npm run dev
    ```
